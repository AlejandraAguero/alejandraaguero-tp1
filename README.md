TP1 – Curriculum Vitae

Este trabajo práctico consistió en el diseño y desarrollo de un sitio web personal de una sola pagina, aplicando HTML5 y CSS. En la parte de diseño intenté mantener una estructura limpia y ordenada de los elementos, y a su vez aplique un branding personal para todo el desarrollo del sitio.

Editor de código: Visual Studio Code
Tamaño de pantalla utilizado: 3840 × 2160 px (4K) “Resolución de pantalla recomendada”
